The data seems to be normally distributed and can be used for further statistical analysis to determine difference of means. The statistical test used will be a two sample t-test. Null hypothesis is that there is no statistically significant difference in means between the data in outage seconds per week in the churn_yes data frame and the churn_no data frame.the Alternate Hypothesis is that there is a statistically significant difference in the meansbetween the data in outage seconds per week in the churn_yes data frame and the churn_no data frame. the confidence interval of 95 percent, or an Alpha of 0.05¶

Two Sample T-Test¶

In [6]:
#run the t-test from stats and compare the p-value to Alpha of 0.05
#if pval > 0.05 then reject the null hypothesis, if pval < 0.05 then fail to reject the null hypothesis

tstat, pval = stats.ttest_ind(churn_yes.Outage_sec_perweek, churn_no.Outage_sec_perweek)
print(pval)
